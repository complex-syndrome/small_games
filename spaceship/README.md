# Spaceship Game
A game where you drive a spaceship to hit asteroids, as simple as that.
Just an exercise for training game development skills.

# Keys
- w / top arrow     -- Up
- a / left arrow    -- Left
- s / down arrow    -- Donw
- d / right arrow   -- Right
- p / Esc           -- Pause
- r                 -- Restart
